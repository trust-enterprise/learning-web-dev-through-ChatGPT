Flex Vs Grid Learnings

Flexbox
container props (display, flex-direction, justify-content, NO JUSTIFY ITEMS, align-items (for movement of all rows (separated)), align-content (for spacing between & spaced-movement of all rows(consolidated)), gap, wrap)
item props (order, flex (for an item to occupy multiple columns), flex-grow, flex-shrink, align self, justify self)
Grid
container props (all justify & item property)
item props (grid-area (when grid template area is applied to container vid 39 time 21:00), grid column & grid row for overlapping, align self, justify self)
